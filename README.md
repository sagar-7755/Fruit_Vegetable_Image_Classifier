🍎 Fruit & Vegetable Image Classifier

A deep learning-based web application that classifies images of fruits and vegetables using a Convolutional Neural Network (CNN). The model is built using a Sequential architecture and deployed with an interactive UI using Streamlit.

⸻

🚀 Project Overview

This project focuses on building an image classification system capable of identifying different fruits and vegetables from images. It leverages deep learning techniques and provides a simple, user-friendly interface where users can upload an image and get predictions instantly.

⸻

🛠️ Tech Stack
	•	Python
	•	NumPy – Numerical computations
	•	Pandas – Data handling and preprocessing
	•	TensorFlow & Keras – Model building and training
	•	Jupyter Notebook – Model development and experimentation
	•	VS Code – Development environment
	•	Streamlit – Web app deployment

⸻

🧠 Model Architecture
	•	Built using Keras Sequential Model
	•	Convolutional Neural Network (CNN) for image classification
	•	Layers used:
	•	Convolutional Layers (Conv2D)
	•	Activation Function (ReLU)
	•	Pooling Layers (MaxPooling2D)
	•	Fully Connected Layers (Dense)
	•	Output Layer with Softmax
